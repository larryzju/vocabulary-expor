# vocabulary.bing

从 http://dict.bing.com 查询单词并获取释义，可导出成 HTML 文件

## Usage

```shell
$ lein uberjar
$ java -jar target/uberjar/vocabulary.bing-0.1.0-SNAPSHOT-standalone.jar <words.txt>
```
